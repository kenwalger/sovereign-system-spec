# sovereign-sdk

The Sovereign SDK is the primary reference implementation of the Sovereign Systems Specification.

It provides practical implementations of Sovereign Systems concepts including provenance-aware ingestion, cryptographic custody, deterministic memory boundaries, and forensic auditability.

## Purpose

The SDK exists to transform the Sovereign Systems Specification from architectural guidance into executable software.

Rather than serving as a general AI framework, the SDK focuses on enforcing trust, provenance, and custody throughout the AI execution lifecycle.
